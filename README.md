## 项目介绍

为了更好了整合本人的技术栈，也为了能够帮助到大家学习go语言，尽快掌握go语言的工程实践。

我通过一个简单的业务demo，也就是用户信息的增删查改，来实现一个golang后端API应用。

主要涵盖了下面这些技术点：
* 路由
* 过滤器
* 日志
* 静态配置文件
* 控制器
* 模型
* redis缓存
* MySQL数据库

以后可能会增加更多的技术内容进来，比如etcd, nsq等内容，也欢迎大家一起来完善本项目，造福更多的新人朋友。
